## ЁЯза Part 1 тАУ Train the Model

```bash
python train.py
```

**What it does:**
- Loads `ipl.csv` (real) **or** auto-generates a synthetic dataset
- Encodes teams with `LabelEncoder`, scales numerics with `MinMaxScaler`
- Builds ANN: `Input тЖТ 512 тЖТ 256 тЖТ 128 тЖТ 64 тЖТ 1`
- Trains with Huber loss + EarlyStopping + ReduceLROnPlateau
- Saves `ipl_model.h5` and `artifacts/*.pkl`

**Real dataset (optional):**
Download from [Kaggle тАУ IPL Dataset](https://www.kaggle.com/datasets/manasgarg/ipl)
and place `ipl.csv` in the project root before running.

## ЁЯМР Part 2 тАУ Run the Streamlit App

```bash
streamlit run app.py
```

Open `http://localhost:8501` in your browser.

**Features:**
- Select batting & bowling teams from a dropdown
- Input current score, overs, wickets, and last-5-overs stats
- Real-time run rate, balls left, and wickets-in-hand display
- Predicted score + ┬▒10 run range with colour-coded verdict
- Expandable breakdown panel

## ЁЯЪА Part 3 тАУ Deployment

### Streamlit Community Cloud (free)

1. Push all files to a **public GitHub repo**
2. Go to [share.streamlit.io](https://share.streamlit.io) тЖТ New app
3. Select your repo, set **Main file** = `app.py`
4. Click **Deploy**

> **Important:** `ipl_model.h5` and `artifacts/` must be committed to the repo.

### Render / Railway / Hugging Face Spaces

Add a `Procfile`:
```
web: streamlit run app.py --server.port=$PORT --server.address=0.0.0.0
```

## ЁЯПЧ ANN Architecture

```
Input (7 features)
  тФВ
  тЦ╝
Dense(512, ReLU) тЖТ BatchNorm тЖТ Dropout(0.30)
  тФВ
  тЦ╝

  тФВ
  тЦ╝

  тФВ
  тЦ╝
Dense(64, ReLU)
  тФВ
  тЦ╝
Dense(1, Linear)   тЖР Predicted final score (regression)
```

| Hyperparameter | Value               |
|----------------|---------------------|
| Loss           | Huber               |
| Optimizer      | Adam (lr = 1e-3)    |
| Batch size     | 64                  |
| Max epochs     | 150 (early stop)    |
| Patience       | 15 epochs           |

## тЭУ Troubleshooting

| Error | Fix |
|-------|-----|
| `ipl_model.h5 not found` | Run `python train.py` first |
| `artifacts/*.pkl not found` | Run `python train.py` first |
| TensorFlow import slow | Normal on first run; subsequent runs are faster |
| `ValueError: Unknown team` | The encoder falls back to index 0 safely |
